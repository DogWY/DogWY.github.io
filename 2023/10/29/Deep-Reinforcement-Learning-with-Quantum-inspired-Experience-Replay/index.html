Deep Reinforcement Learning with Quantum-inspired Experience Replay

Abstract

本文提出了一种受量子计算启发的新颖的训练泛式，来训练基于经验回放的强化学习。

与传统的深度学习中的经验回放机制相比，本文提出的基于量子启发经验回放的深度强化学习方法（DRL-QER）根据每个经验（也叫转移）的复杂性和和回放次数来自动从回放缓冲区选择经验，进而实现在探索和利用之间的平衡。

在DRL-QER中，转移首先用量子形式阐述，并且之后对转移操作进行准备操作和折旧操作。

在这个步骤中，准备操作反应了时序误差和经验重要性的关系，同时折旧操作被用来确保转移的多样性。

此方法在Atari2600的大多数游戏中的表现表明DRL-QER比最先进的算法（比如DRL-PER和DCRL）表现还要好，同时提升了训练效率， 并且也可以被应用在记忆基础的DRL方法中作为双网络和决斗网络。

关键词：深度强化学习，量子强化学习，量子计算，受量子启发的经验重播

Introduction

介绍强化学习，引出DQN

强化学习是一个从与环境的交互中学习的智能泛式。

在交互的算法的训练过程中，与人类的行为类似，智能体根据从环境中收到的反作用，调整它的行为来最大化整个控制任务的累计奖励。

当面对真实世界的一般情况时，大多数的控制任务经常会伴随高纬度的输入，此时传统的强化学习方法往往不会有好的表现。

幸运的是，深度学习提供了一个新的方式来处理复杂的输入信息并且已经在各种领域取得了巨大的突破。

特别的是，通过将深度学习整合到强化学习中，一个新的深度强化学习框架出现了，现在深度Q网络（DQN）已经变成了最受欢迎的深度强化学习方法之一。

DQN被用来评估动作价值来帮助智能体进行顺序决策，此方法中，原始图片被喂入带有全链接层的神经网络中来估计未来奖励的估计值。

介绍DQN中的经验回放机制，并简单分析几个受欢迎的经验回放方法的变种。

为了提升状态动作转换的利用率，经验回放的机制被应用到DQN框架中；这种方法中，经验被存储在一个有限大小的缓冲区中，并且从缓冲中被检索。

经验回放的机制有效的加速了训练时的经验处理，但是它忽略了经验之间不同的重要性。

一系列的经验回放变体已经被提出，用来改进学习过程，比如优先级经验回放（PER），深度课程强化学习（DCRL），经验回放的记忆与遗忘（ReF-ER），注意力经验回放（AER）和竞争经验回放（CER）。

在PER中，时序误差决定了经验的优先级并影响经验被回放的可能性。

与原始的经验回放相比此方法（PER）进一步增强了对经验的利用率， 但是智能体生成的数据是有噪声的。

因此DRL-PER给予了时序误差较大的转移更高的优先级并且可能存在着一些经过多次回放，仍不会降低自身较大时序误差的经验。

考虑到这种情况，DRL-PER可能导致了一些经验被过渡使用了，这可能导致神经网络的震荡。

为了提升DQN的采样效率，DCRL提出了一个采样重要性的标准，这个重要性是基于经验的困难和多样性的；困难和时序误差是正相关的，并且多样性于回放次数相关。

然而，它引入了一定的参数，这些参数要求更多的先验知识来准确的调整。

在ReF-ER中，根据Kullback-Leribler分期策略更新会被惩罚，从而加速收敛；注意力经验回放根据他们的状态和只能难题的状态的相似性挑选经验；CER设置了两个智能体来竞争地探索。

但是这三个方法高度的依赖计算资源。

因此，我们需要一个更加高效和通用的方法来增强DRL的经验回放方法。

与此同时，量子物理的引入已经极大的提升了信息处理能力并且对应用人工智能的特定算法任务起到了积极的影响。

特别的是，人们已经对量子强化学习和他们的应用很感兴趣了。

量子强化学习起初来源于将量子并行特征引入经典的强化学习算法中，这实现了一个在探索和利用之间更好的权衡，并且也加速了学习的速度。

人们发现量子力学能够为智能体带来整体二次方的加速。

常见的智能体环境框架也被扩展到了量子领域。

此外，多量子位的量子强化学习被在超导电路上评估，并且被扩展到了其他的案例，比如多级和开放量子系统。

使用Grover算法的多价值函数被证明能够比经典算法更快的收敛。

最近的研究也证实了使用Boltzmann机的强化学习方法的比起经典方法是有优势的。

受量子机器学习的启发，我们可能从数据中产生非典型的模式。

例如，量子叠加态在n个量子位的线性物理空间中提供了指数规模的计算空间。

本文章中，我们为深度强化学（DRL-QER）提出了一个量子启发的经验回放方法，来提升DRL的训练表现，这种方法是自然的，无需超参数调节的。

在DRL-QER中，经验以量子形式表达，并且经验的量子表示的肯能行振幅被通过量子操作迭代的控制，量子操作包括准备操作和折扣操作。

特别的是，准备操作根据经验重要性设计，并且折扣操作与被选经验的回放次数相关。

在这两种操作下，经验的重要性是显而易见的，并且经验的多样性是有保障的。

为了测试提出的DRL-QER算法，我们在Gym-Atari平台上进行了实验，并且比较了DRL-PER和DCRL。

此外，DRL-QER以双重DQN和决斗DQN方式实现，并且DRL-QER也与他对应的经典方法进行了比较。

II. Preliminaries

A. Deep Reinforcement Learning and Experience Replay

Markov Decision Process

RL的训练过程是基于马尔科夫决策模型，它可以被描述为\(<S,A,P,R>\)，S指状态空间，A是动作空间，\(P:S\times A \times S \rightarrow [0,1]\)是状态转移概率，并且\(R:S\times A\rightarrow \mathbb{R}\)是奖励函数。

在和环境的交互中，智能体从\(t\in[0,T]\)时刻采样得到\(s_t\in S\)状态，并且选择一个动作\(a_t = \pi(s_t), a_t\in A\)，策略\(\pi\)是状态\(S\)到动作\(A\)的映射。

在执行完动作\(a_t\)后，智能体转移到下一个状态\(s_{t+1}\)并且收到一个标量奖励信号\(r_t\)。因此我们得到一次在时间\(t\in T\)的转移\(e_t = (s_t ,a_t,r_t,s_{t+1})\)。

RL的目标是一个能让未来累计折扣奖励\(\sum\limits_{k=0}^{T-t} \gamma^k r_{t+k}\)最大的策略\(\pi^*\)，其中\(\gamma\in[0,1]\)是一个用来平衡当前奖励重要性和未来奖励重要性的折扣因子。

随着RL算法的广泛使用，QLearning定义\(Q(s,a)\)作为状态\(s\)下遵从策略\(\pi\)执行动作\(a\)期望折扣奖励，并且建立一个Q值的查找表来查找Q值。

Deep Q Network

在高纬度环境中，使用带有参数\(\theta\)参数的神经网络来近似\(Q(s,a)\)是一个普遍且有效的方法， \(Q(s,a;\theta) \approx Q(s,a)\)，而不是用一个查询表来存储所有的状态动作价值\(Q(s,a)\)。

为了基于梯度下降法来更新神经网络的参数，状态动作价值\(Q(s,a)\)的“真实值”\(y(s,a)\)根据最大化下一个动作价值函数\(y(s,a) = r + \gamma \max\limits_{a'}Q(s',a';\theta^-)\)来进行估计。其中\(\theta^-\)代表目标网络的参数，这些参数在计算\(y(s,a)\)期间是固定的，并且在一些训练步骤之后更新。

时序误差(TD-errors)\(\delta\)可以根据\(y(s,a)\)和\(Q(s,a)\)之间差值获取： \[ \delta = y(s,a) - Q(s,a) = r + \gamma \max\limits_{a'}Q(s',a';\theta^-) - Q(s,a;\theta) \] 根据，损失函数进行优化： \[ Loss(\theta;Q,y) = \frac{1}{2} (r + \gamma \max\limits_{a'} Q(s',a';\theta') - Q(s,a;\theta))^2 \] 讲损失函数对参数\(\theta\)参数进行求导，得到如下梯度： \[ \nabla_\theta Loss = [r + \gamma \max\limits_{a'} Q(s',a';\theta^-) = Q(s,a;\theta)]\nabla_\theta Q(s,a;\theta) \]

Experence Replay

在大多数的RL框架中，智能体在观察到一个系列的经验后逐渐更新他们的参数。

在最简单的形式中，输入数据用来进行单步更新并且立刻丢弃，这带来了两个缺点：

  • 强相关转变打破了独立同分布假设，这对许多流行的基于随机梯度的算法来说是必要的假设
  • 有些罕见的经验会被快速的遗忘，这可能会在未来导致低效的采样

一个自然的想法是，将过去的经验放到一个大的缓存中，并且从其中选出一个组经验来训练。

这种步骤被称为经验回放。

在经验回放中，如何选择经验来回放是提升DRL训练表现的一个至关重要的因素。

当我们将一个转移（或者说经验）\(e_t\)放到一个固定的大小为\(M\)的经验回放缓存中， 一个新的索引标签\(k\in\{1,\dots,M\}\)被分配给它， 同时它的优先级被表示为\(P_k\)。

像这样，整个经验缓存能够被视为一个转移集合\(\{<e_t,P_k>\}\)。

一个完整的经验回放过程实际上是一个存储-采样的过程，并且学习过程是通过从缓存中挑选一个小规模采样来更新RL智能体参数实现的。

经验回放的关键是如何衡量每一个转移的重要性，或者说如何界定每一个转移的\(P_k\)。

Quantum Computation

在量子计算中，携带信息的基础单元是一个量子比特，并且一个量子比特能够处于基本征态\(|0>\)和\(|1>\)的叠加态，即可以被表示为如下形式： \[ |\psi> = \alpha |0> + \beta |1> \] 其中\(\alpha\)和\(\beta\)满足\(|\alpha|^2 + |\beta|^2 = 1\)。

量子力学表明，测量处于叠加态 \(|\psi>\) 的量子位会导致其以 \(|\alpha|^2\)的概率塌陷到\(|0>\)的本征态之一，或以\(|\beta|^2\)的概率塌陷到\(|1>\)的本征态之一。

特别的是，系数能够被写作\(\alpha = <0|\psi>\)和\(\beta = <1|\psi>\)，其中\(<\alpha|\beta>\)表示\(|\alpha>\)和\(|\beta>\)的内积。

在量子计算中，酉变换是一个量子系统中的基础操作，并且能够转移一个厨师状态\(|\psi>\)到另一个状态\(|\psi'>\): \[ |\psi'> = U|\psi> \] 其中\(U\)满足\(U^{\dagger}U = UU^{\dagger} = I\)。

例如， 一个将\(|0>\)转移到\((|0> + |1>)/\sqrt{2}\)或将\(|1>\)转移到\((|0> - |1>)/\sqrt{2}\)的Hadamard门，可以被表述为： \[ H = \frac{1}{\sqrt{2}} \left[ \begin{array} \ 1\ \ \ 1\\ \ 1 -1 \end{array} \right] \] 另一个重要的量子们是相位门，他是执行Grover迭代来增强“目标”幅度的重要元素。

关于量子操作和量子门的更多讨论可以在38文献中找到。

Grover算法是最重要的量子算法之一。

他已经被广泛使用在大规模数据库搜索中， 并且它在非结构数据库中有较高的可能能在\(O(\sqrt{N})\)的复杂度下定位元素。

他的核心概念是将元素以量子系统表示， 并且使用一个酉操作以迭代的方式操作其状态。

作为 Grover 算法的主要运算之一，Grover迭代已经成功的在RL中得到了应用，其中动作以可能的特征动作的叠加态表示。

然后，在叠加态上，酉转移迭代的被执行来改变“好”行为的概率幅度。

复合量子系统的状态空间由张量积表示，表示为\(\otimes\)。

例如，两个子系统\(A\)和\(B\)组成的复合量子系统可以在Hilbert空间上被定义为\(\mathcal{H} = \mathcal{H}_A \otimes \mathcal{H}_B\)， 其中\(\mathcal{H}_A\)和\(\mathcal{H}_B\)分别表示\(A\)和\(B\)在Hilbert空间中的表示。

此外，他的状态\(|\psi>\)可以被表示为子系统状态的张量积\(|\psi_{AB}> = |\psi>_A \otimes |\psi>_B\)。

为了通过测量和观测一个量子系统来获取信息， 可以应用POVM(positive-operator-valued measure)。

对于一个可观测的\(H\)，存在一套完整的正交投影\(\{X_h : \sum_h X_h = I, X_h = X_h^{\dagger},X_{\hat{h}}X_h = \delta_{\hat{h},h}, X_h \}\)， 其中\(\delta_{\hat{i},i}\)是Kronecker delta， \(X_h\)是特征值 \(h\) 到 \(H\) 特征空间的投影，我们有 \(H = \sum_h hX_h\)。获得结果 \(h\) 的概率可以通过\(P(h) = <\psi|X_h|\psi>\)计算。

Conclusion

本文通过将量子特征引入DRL中的经验充播过程中，提出了DRL-QER方法，此方法能够保证我们的学习会关注于智能体在和环境交互中学到的东西而不是先验知识。

在DRL-QER中，经验被量子状态表示，他的振幅和时序误差与回放次数相关。

特别的是，DRL-QER中的准备操作和折旧操作帮助加速了训练过程并且提升了采样效率。

时间结果证实了DRL-QER的表现比DRL-PER和DCRL更加优越。

DRL-PER和DRL-QER在决斗DQN和双重DQN中的比较展示出DRL-QER也能够提升其他的基于记忆的DRL算法的表现。

我们的未来的工作将会聚焦于DRL-QER的收敛问题的深入研究，及其在其他连续控制方法中的应用，例如深度确定性策略梯度（DDPG）和软演员评论家。
